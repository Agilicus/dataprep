DataPrep Documentation¶

Version: 0.4.0

DataPrep lets you prepare your data using a single library with a few lines of code.

Documentation | Discord | Forum

Currently, you can use DataPrep to:

  • Collect data from common data sources (through Connector)

  • Do your exploratory data analysis (through EDA)

  • Clean and standardize data (through Clean)

  • …more modules are coming
